### [FreshRSS](https://freshrss.org)

#### Install using Git

If you are a git user, you can install the theme and keep up to date by cloning the repo:

```bash
cd /path/to/FreshRSS/p/themes
git clone https://github.com/dracula/freshrss.git
```

#### Install manually

Download using the [GitHub `.zip` download](https://github.com/dracula/freshrss/archive/main.zip) option and unzip them into the `p/themes` directory of your FreshRSS installation.

#### Activating theme

1. Log in to your FreshRSS instance;
2. Go to **Configuration** (⚙️ gear icon) → **Display**;
3. Under **Theme**, select **Dracula**;
4. Scroll down and click **Submit** to save;
5. Boom! It's working ✨
